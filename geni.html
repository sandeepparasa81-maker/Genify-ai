🚀 Deploy This Site To Netlify

Host your own live, public content generation workspace in less than 2 minutes for free.

1
Save This Code: Save the current code to a file named index.html.
2
Paste Your Key (Optional): To share the live generation with everyone who visits, open your index.html file in any text editor and paste your Google AI Studio API Key in the GLOBAL_API_KEY field at the very beginning of the script tag.
3
Zip & Drag: Put the index.html file inside an empty folder, zip it, and drag it into Netlify's manual deploy box at app.netlify.com.
⚠️ CORS Note: Unlike OpenAI or Claude APIs, Google's Gemini API permits direct browser calls out-of-the-box. This ensures your Netlify hosted site can execute content generations with zero CORS blockages!
